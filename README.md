Modern, scalable, and production-ready frontend architecture built with **React 18, Vite, TypeScript, Tailwind CSS, and modern UI primitives**.

Designed for performance, maintainability, and seamless deployment workflows.

---

## ✨ Features

- ⚡ **React 18 + Vite 6 + TypeScript 5**
- 🎨 **Tailwind CSS** + Radix UI primitives
- 🧩 Reusable **shadcn-style UI components**
- 🛣️ Client-side routing using **React Router**
- 📦 Code splitting with **React.lazy**
- 🔔 Toast notifications
- 📝 Form handling with **react-hook-form**
- ✅ Schema validation with **zod**
- 🚀 Optimized frontend architecture
- 🎬 Smooth animations using **GSAP**
- 🐳 Dockerized deployment support
- 🌐 SPA-ready **Nginx configuration**
- ☁️ Deployment support for **Vercel / AWS ECS / Docker**
